## ğŸ¯ Overview

The Hall Booking System is a web-based application designed to streamline the process of reserving halls and facilities managed by Majlis Daerah Kuala Krai. The system provides an intuitive interface for users to book halls, select additional equipment, and manage their reservations efficiently.

### Key Benefits

- **Digital Transformation**: Move from manual booking processes to automated digital system
- **Real-time Availability**: Check hall availability in real-time
- **Automated Pricing**: Calculate costs automatically based on hall and equipment selection
- **Document Management**: Handle supporting documents digitally
- **User-friendly Interface**: Modern, responsive design accessible on all devices
